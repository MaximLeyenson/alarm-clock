**Installation for one user**
```bash
   $ mkdir -pv ~/bin
   $ cd ~/bin/
   $ git clone https://gitlab.com/maxim.leyenson/alarm-clock
```


and then add the lines

   ```
   PATH=$PATH:$HOME/bin/alarm-clock
   export PATH
   ```
to your .bashrc
